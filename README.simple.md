# AuroraBot

Fast, smart, easy, fast way to interact with Aurora OS.

![preview](data/preview.png)

### Features

- `cli` - Classic command line.
- `cmd` - Smart command line.
- `faq` - Answers on the Aurora OS ecosystem.
- `svc` - Services: dbus api, telegram client.
